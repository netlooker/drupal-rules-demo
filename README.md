### Rules demo feature

Main goal is to show for all of those who don't know capacity of Rules module how to: 
  - define events, 
  - define conditions, 
  - define actions,
  - trigger events.

This material is made mainly for showcasing and it was prepared for the "Drupaleuven bijeenkomst Oktober 2015"
meeting.
